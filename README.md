# Monitoramento de URLs
Script desenvolvido em Golang para Monitoramento de URLs, Status Code e exibição de logs.

## Exemplo em Go
```console
$ go run monitor.go

1. Monitoring Url
2. Viewing Logs
0. Exit        
Options:   
```
## URLs
```console
$ cat urls.txt
https://alura.com.br       
https://caelum.com.br      
https://casadocodigo.com.br
```
